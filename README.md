## ğŸš€ Getting Started

### Prerequisites
- Python 3.7 or higher
- Jupyter Notebook or JupyterLab
- pip or conda package manager
- Basic Python knowledge

### Installation

**Step 1: Clone the repository**
```bash
git clone https://github.com/surag29/VISUALIZATION-LIBRARIES-REPO.git
cd VISUALIZATION-LIBRARIES-REPO
```

**Step 2: Install dependencies**
```bash
pip install jupyter pandas numpy matplotlib seaborn plotly cufflinks
```

Or using conda:
```bash
conda install jupyter pandas numpy matplotlib seaborn plotly
conda install -c conda-forge cufflinks
```

**Step 3: Launch Jupyter Notebook**
```bash
jupyter notebook
```

**Step 4: Open and explore**
Select any `.ipynb` file and start learning!

## âš ï¸ Common Issues & Solutions

**Issue:** Plots not displaying in Jupyter  
**Solution:** Add this at the beginning of your notebook
```python
%matplotlib inline
```

**Issue:** Missing dependencies  
**Solution:** Install all required packages
```bash
pip install -r requirements.txt
```

**Issue:** Plotly visualizations not rendering  
**Solution:** Use nbformat upgrade
```bash
pip install --upgrade nbformat
```
